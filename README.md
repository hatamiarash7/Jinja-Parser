# Jinja Parser

It's a Live parser for Jinja2

## Install

### Local

```bash
git clone https://github.com/hatamiarash7/jinja-parser.git
make install
make run
```

### Docker

```bash
docker run -d -p 8080:8080 hatamiarash7/jinja-parser
```
